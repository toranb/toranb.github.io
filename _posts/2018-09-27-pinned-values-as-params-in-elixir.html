In part 4 of my Elixir journey I wanted to show how pinned values work when they are used with functions.

Pinned Values

One example I found just fascinating in Elixir is how the pinned operator works along with matching when used as part of the function params.

My shallow understanding of the pin operator before using them with functions was simply "do not re-bind this variable". In functions it works much the same way but in the example above you begin to see the real power of pattern matching. The 2nd function having just an underscore acts like a default implementation. Using pattern matching and the pinned operator helps avoid what would otherwise be a conditional.
